How to Setup REVENTOR DNS
REVENTOR DNS provides comprehensive protection against ads, trackers, malware, and unwanted content. Enjoy faster browsing, enhanced privacy, and parental controls across all your devices.

Your DNS server addresses:
192.168.1.100

dns.reventor.eu
Choose your device type below for setup instructions. Use the IP address for most devices, or the domain name for private DNS settings on mobile devices.
Router Setup
  1. Open your router's web interface (usually 192.168.1.1 or 192.168.0.1)
  2. Log in with your router credentials
  3. Navigate to Network > DHCP or WAN settings
  4. Find DNS server settings and set to: 192.168.1.100
  5. Save changes and reboot router if required
Note: This will protect all devices on your network automatically.
Windows Setup
  1. Right-click Start button > Settings > Network & Internet
  2. Click Properties under your network connection
  3. Scroll to IP settings > Edit
  4. Change to Manual and set Preferred DNS to: 192.168.1.100
  5. Save the changes
macOS Setup
  1. Open System Settings > Network
  2. Select your Wi-Fi or Ethernet connection
  3. Click Details > DNS
  4. Click the + button and add: 192.168.1.100
  5. Click OK to save
Android Setup
  1. Open Settings > Network & internet > Advanced > Private DNS
  2. Select "Private DNS provider hostname"
  3. Enter: dns.reventor.eu
  4. Tap Save
Note: Requires Android 9+ and a DNS-over-TLS compatible server.
iOS Setup
  1. Open Settings > Wi-Fi
  2. Tap the info (i) icon next to your Wi-Fi network
  3. Scroll to DNS section and tap Configure DNS
  4. Select Manual and add: dns.reventor.eu
  5. Tap Save
Note: For encrypted DNS, ensure your server supports DNS-over-HTTPS.
